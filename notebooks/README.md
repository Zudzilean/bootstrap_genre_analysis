# Analysis Notebooks

This directory contains Jupyter notebooks for the bootstrap analysis project.

## Notebook Structure

1. **01_data_preprocessing.ipynb**: Data loading, cleaning, and transformation
2. **02_bootstrap_analysis.ipynb**: Bootstrap resampling and statistical analysis
3. **03_visualization.ipynb**: Creating plots and figures
4. **04_integration.ipynb**: End-to-end workflow combining all modules

## Usage

1. Start Jupyter: `jupyter notebook`
2. Run notebooks in order (01 → 02 → 03 → 04)
3. Ensure data is in `../data/processed/` before running analysis notebooks
